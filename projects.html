Portfolio Website

Weather App*

This website was built from scratch as my final project on the SheCodes Plus course, using vanilla Javascript (updated React-based version coming soon!), HTML and CSS. Using an integration to the OpenWeather API, the project features a search engine, current location weather functionality, and daily forecast. The application was built via a typical DevOps workflow using VSCode, Git, GitHub and Netlify.

*Please bear in mind the project is currently not responsive.
